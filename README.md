# ARIA v2.0 — Terrain Intelligence Upgrade

### Project Overview
This project extends river proximity analysis by integrating Digital Elevation Model (DEM) data to assess terrain risks (slope and elevation) for evacuation shelters in Hualien.

### Deliverables
- **ARIA_v2.ipynb**: Full analysis workflow.
- **terrain_risk_audit.json**: Quantitative risk metrics for each shelter.
- **terrain_risk_map.png**: Final visualization of elevation and slope risks.

### Key Findings
Shelters located in higher elevations (e.g., SH-光復鄉-4) show significantly higher slope risks (max slope > 50°), indicating a need for enhanced landslide monitoring.
